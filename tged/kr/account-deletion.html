계정 삭제

Detail Games 계정을 삭제하고 싶으시다면, 아래 안내에 따라 진행해 주세요.

⚠️ 주의사항

계정 삭제는 되돌릴 수 없는 작업입니다. 계정 삭제 시 다음 데이터가 영구적으로 삭제됩니다:

  • 프로필 정보 및 설정
  • 게임 진행 상황 및 저장 데이터
  • 구매 내역 및 결제 정보
  • 친구 목록 및 소셜 기능 데이터
  • 기타 모든 개인 데이터

계정 삭제 절차

계정 삭제를 위해서는 다음 단계를 따라주세요:

1단계: 이메일 발송

아래 이메일 주소로 계정 삭제 요청을 보내주세요:

계정 삭제 요청 이메일 주소:

dev01@detailgames.ai

제목: [계정 삭제 요청] 사용자명 또는 이메일

2단계: 이메일 내용

이메일에 다음 정보를 포함해 주세요:

  • 제목: [계정 삭제 요청] 사용자명 또는 이메일
  • 본문:
    • 삭제를 요청하는 계정의 이메일 주소
    • 사용자명 (닉네임)
    • 계정 삭제 사유 (선택사항)
    • 계정 삭제에 대한 동의 확인

3단계: 처리 기간

계정 삭제 요청을 받은 후, 다음과 같은 기간 내에 처리됩니다:

  • 영업일 기준 3-5일 내에 계정 삭제 처리
  • 처리 완료 시 이메일로 확인 통지
  • 긴급한 경우 별도 문의 가능

계정 삭제 전 확인사항

계정 삭제 전에 다음 사항을 확인해 주세요:

  • 중요한 게임 데이터가 백업되었는지 확인
  • 진행 중인 결제나 구독 서비스가 있는지 확인
  • 다른 서비스와 연동된 계정이 있는지 확인
  • 계정 삭제 후 복구가 불가능함을 이해했는지 확인

계정 삭제 후

계정이 삭제된 후에는:

  • 해당 계정으로 서비스에 로그인할 수 없습니다
  • 모든 개인 데이터가 영구적으로 삭제됩니다
  • 동일한 이메일로 새 계정을 만들 수 있습니다
  • 삭제된 데이터는 복구할 수 없습니다

대안: 계정 비활성화

완전한 계정 삭제 대신, 일시적으로 계정을 비활성화하는 방법도 있습니다:

  • 계정 비활성화 시 로그인은 불가능하지만 데이터는 보존됩니다
  • 나중에 다시 활성화하여 계속 사용할 수 있습니다
  • 계정 비활성화를 원하시면 동일한 이메일로 문의해 주세요
