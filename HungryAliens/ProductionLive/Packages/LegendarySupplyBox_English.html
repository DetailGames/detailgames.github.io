Legendary Supply Box guaranteed rate

If the target appears within the set number of attempts, the counter resets.

If the target does not appear by the set number of attempts, it is guaranteed to appear on the final attempt.

If two different types are guaranteed at the final attempt, one will appear, and the other will be postponed to the next round.
